Cara pakai:
1. Install Texstudio (Latex editor)
2. Install MikTex (semacam bahasa pemrogramannya Latex gitu)
3. Buka Texstudio:
	- Open file "skripsi.tex" -> buka file ini saja (karena ini semacam induknya)
	- Kalau ada popup yg minta install sesuatu, install aja dulu, mungkin butuh library tertentu
4. Edit-edit deh ikutin yg udah ada aja
5. compile / create pdf klik tombol ">>"

Atur language checking:
- Buka menu "options" > "configure TexStudio" > "Language Checking" > ganti default language nya, kalau gaada download aja di http://extensions.openoffice.org/de/project/indonesian-dictionary-kamus-indonesia


Referensi:
- kalau bisa referensi ambil langsung citex nya saja, gausah repot bikin sendiri
- copas cites taro di "skripsi.bib"
- referensi ini agak ribet kalau udah error, penyebab error biasanya jika citex nya tidak ada 'year'
- kalau sudah error, hapus citex yg bikin error, hapus dulu file skripsi.aux, skripsi.bbl, skripsi.blg, skripsi.lof, skripsi.log, skripsi.lot, skripsi.out, skripsi.toc, compile lagi deh
